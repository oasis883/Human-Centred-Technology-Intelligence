# OAuth, SAML and Token-Based Authentication

This project uses token-based authentication practically and documents OAuth/SAML concepts.

## Token-based auth
Examples: Claude API key, Slack webhook URL, GitHub token, Azure DevOps token.

## OAuth
Allows an app to access another app without sharing passwords.

## SAML
Used for enterprise Single Sign-On through Microsoft Entra ID, Okta, Slack, GitLab, Freshservice and Atlassian.
