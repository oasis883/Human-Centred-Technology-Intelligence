# HumanTech Intelligence Report

## Summary

Total tickets analysed: **50000**

## Behaviour Categories

- **Memory / Access**: 16011 tickets (32.0%)
- **Communication / Collaboration**: 14670 tickets (29.3%)
- **General IT Support**: 13121 tickets (26.2%)
- **Mental Model**: 4211 tickets (8.4%)
- **Attention / Security**: 1056 tickets (2.1%)
- **Training Gap**: 635 tickets (1.3%)
- **Technology Anxiety**: 296 tickets (0.6%)

## Practical Improvements

- Simplify login, improve MFA/password guidance, consider SSO or passwordless options. (16011 related tickets)
- Improve Teams, Outlook and meeting-room support documentation. (14670 related tickets)
- Review recurring patterns and improve documentation. (13121 related tickets)
- Create simple visual guides explaining how the system works. (4211 related tickets)
- Improve cyber awareness, warning messages and phishing reporting guidance. (1056 related tickets)
- Create targeted training and self-service knowledge articles. (635 related tickets)
- Improve communication, reassurance and outage/status updates. (296 related tickets)

## Project Meaning

This report shows how IT tickets can reveal human behaviour patterns such as memory load, technology anxiety, mental model gaps, training needs and security awareness.

The aim is not to blame users. The aim is to improve systems, documentation, communication and employee experience.
