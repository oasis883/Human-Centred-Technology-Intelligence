# Human-Centred Technology Intelligence

AI-powered behavioural analytics for IT support tickets.

## Overview

This project analyses synthetic IT support ticket data to identify human-centred technology patterns such as memory load, communication friction, mental model gaps, training needs, and security awareness.

## Workflow

```text
Synthetic Tickets
→ Python Data Cleaning
→ Behaviour Classification
→ Insights Report
→ Claude AI Summary
→ Slack Notification
→ Streamlit Dashboard
→ GitHub Actions
