# 🧠 Human-Centred Technology Intelligence

> AI-powered behavioural analytics for IT support operations using Python, Claude AI, Streamlit, Slack and GitHub Actions.

---

## 📖 Overview

Human-Centred Technology Intelligence is a personal portfolio project that explores how AI and behavioural analytics can improve the employee technology experience.

Rather than treating IT support tickets as isolated technical incidents, this project analyses **synthetic IT support data** to identify recurring behavioural patterns such as:

- 🧠 Memory and access challenges
- 💬 Communication friction
- 📚 Knowledge and training gaps
- 🔒 Security awareness
- ⚙️ Technology friction

The project demonstrates an end-to-end AI workflow that transforms synthetic IT support data into actionable insights through automation, interactive dashboards, and AI-generated executive summaries.
